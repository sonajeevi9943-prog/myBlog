# myBlog
MyBlog is a beginner-friendly blog website built with pure HTML, CSS &amp; JavaScript. It features blog post cards, a sidebar, category filters, and a newsletter — all without any frameworks!
📋 What This Blog Website Contains
1️⃣ Navbar

Logo on the left — "MyBlog"
Menu links — Home, Tech, Life, Projects, About, Contact
Hamburger menu (☰) appears on mobile
Sticky at top while scrolling


2️⃣ Hero Section

Welcome tag — "Welcome to My Blog"
Big headline — Stories, Tips & Life as a Developer
Short description text
Search bar to search articles


3️⃣ Category Filter

6 filter buttons — All, HTML & CSS, JavaScript, Projects, Career Tips, Life & Learning
Click any button to highlight it as active


4️⃣ Featured Post (Big Card)

Large colorful banner image
Post category tag
Post title & description
Author avatar, name & read time
Read More button


5️⃣ Latest Posts Grid (6 Cards)
Each post card has:

Colorful gradient image
Category tag
Post title
Short description
Date, read time & like count ❤️
Hover effect — lifts up with shadow


6️⃣ Sidebar (Right Column)

About Me — avatar, bio & social links (GitHub, LinkedIn, Twitter)
Popular Posts — top 4 posts with like count
Topics/Tags — clickable tag cloud (HTML, CSS, JS, Career, etc.)
Newsletter — name + email input with Subscribe button


HTML5      — Structure & content
CSS3       — Styling, Grid, Flexbox, Animations
JavaScript — Filter, Newsletter, Scroll effects
Google Fonts — Merriweather + Poppins
No libraries — 100% pure beginner code!

Click to load more posts
